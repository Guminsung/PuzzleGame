# 15퍼즐 미니 기획

## 한 줄 소개
- [예: 빈칸을 이용해 타일을 이동, 1~15 정렬 시 클리어]

## 목표/비목표
- 목표: 4x4 보드, 클릭 이동, 해결 가능한 셔플, 이동 수 표시, 리셋
- 비목표: 화려한 연출/사운드, 모바일 최적화(초기 제외)

## 코어 루프
- 클릭 → 유효 이동(빈칸과 인접) → 스왑 → 완료 체크 → 반복

## 규칙/파라미터
- 보드: 4x4, 타일: 1~15 + 빈칸
- 입력: 마우스 클릭
- 셔플: 해결 가능한 순열만 생성
- 승리 조건: 1..15 순서 + 빈칸 마지막

## UI
- 이동 횟수, Reset 버튼

## 기술 선택
- Unity [LTS 버전], 템플릿 2D(Core), 해상도 1920x1080

## 리스크/메모
- 순열 판정 정확도, 추후 모바일/효과 확장
